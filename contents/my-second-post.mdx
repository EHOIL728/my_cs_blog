# Python으로 시작하는 데이터 전처리 가이드

데이터 분석에서 많은 시간은 모델링보다 **전처리**에 쓰입니다.
이 글에서는 실무나 개인 프로젝트에서 자주 마주치는 기본 정리 흐름을 짧게 훑어봅니다.

## 1. 데이터 불러오기

가장 먼저 해야 할 일은 데이터를 안전하게 읽고 구조를 확인하는 것입니다.

```python
import pandas as pd


df = pd.read_csv("data.csv")
print(df.head())
print(df.info())
```

## 2. 결측치 확인하기

결측치는 분석 결과를 왜곡할 수 있기 때문에 먼저 분포를 확인해두는 편이 좋습니다.

```python
missing_count = df.isnull().sum()
missing_ratio = df.isnull().mean() * 100

print(missing_count)
print(missing_ratio)
```

## 3. 기본적인 처리 전략

상황에 따라 아래 전략을 조합합니다.

1. 결측치가 적고 의미가 작다면 제거하기
2. 평균, 중앙값, 최빈값으로 대체하기
3. 모델 기반 추정으로 보완하기

```python
df = df.dropna(subset=["important_column"])
df["score"] = df["score"].fillna(df["score"].median())
```

## 4. 이상치 살펴보기

이상치는 단순한 오류일 수도 있고 중요한 신호일 수도 있습니다.
IQR 방식은 빠르게 전체 분포를 점검할 때 유용합니다.

```python
q1 = df["value"].quantile(0.25)
q3 = df["value"].quantile(0.75)
iqr = q3 - q1

filtered = df[
    (df["value"] >= q1 - 1.5 * iqr)
    & (df["value"] <= q3 + 1.5 * iqr)
]
```
