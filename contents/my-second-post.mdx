# Python으로 시작하는 데이터 전처리 완벽 가이드

데이터 분석의 80%는 데이터 전처리에서 시작됩니다. 이번 포스트에서는 Pandas와 NumPy를 활용한 효율적인 데이터 정제 및 변환 기법을 소개합니다.

## 1. 데이터 불러오기

먼저 Pandas를 사용하여 데이터를 불러오는 방법부터 시작하겠습니다.

```python
import pandas as pd
import numpy as np

# CSV 파일 읽기
df = pd.read_csv('data.csv')

# 첫 5행 확인
print(df.head())
```

## 2. 결측치 처리

결측치는 데이터 분석에서 가장 흔하게 마주치는 문제입니다.

### 결측치 확인

```python
# 결측치 개수 확인
print(df.isnull().sum())

# 결측치 비율 확인
print(df.isnull().mean() * 100)
```

### 결측치 처리 방법

1. **삭제**: 결측치가 있는 행/열 제거
2. **대체**: 평균, 중앙값, 최빈값으로 대체
3. **예측**: 머신러닝 모델로 예측하여 채우기

```python
# 결측치가 있는 행 삭제
df_cleaned = df.dropna()

# 평균값으로 대체
df['column'] = df['column'].fillna(df['column'].mean())

# 특정 값으로 대체
df['column'] = df['column'].fillna(0)
```

## 3. 이상치 처리

이상치는 분석 결과를 왜곡할 수 있으므로 적절한 처리가 필요합니다.

### IQR 방법을 이용한 이상치 탐지

```python
Q1 = df['column'].quantile(0.25)
Q3 = df['column'].quantile(0.75)
IQR = Q3 - Q1

# 이상치 필터링
df_filtered = df[
    (df['column'] >= Q1 - 1.5 * IQR) &
    (df['column'] <= Q3 + 1.5 * IQR)
]
```

## 4. 데이터 타입 변환

적절한 데이터 타입은 메모리 효율성과 분석 정확도를 높입니다.

```python
# 문자열을 숫자로 변환
df['price'] = pd.to_numeric(df['price'], errors='coerce')

# 날짜 형식 변환
df['date'] = pd.to_datetime(df['date'])

# 카테고리 타입 변환 (메모리 절약)
df['category'] = df['category'].astype('category')
```

## 5. 데이터 정규화 및 표준화

머신러닝 모델 학습을 위해서는 스케일링이 필수적입니다.

```python
from sklearn.preprocessing import StandardScaler, MinMaxScaler

# 표준화 (평균 0, 분산 1)
scaler = StandardScaler()
df['scaled'] = scaler.fit_transform(df[['column']])

# 정규화 (0~1 범위)
minmax = MinMaxScaler()
df['normalized'] = minmax.fit_transform(df[['column']])
```
