# MDX 블로그 체크 포스트

이 글은 블로그의 **MDX 렌더링**, **프론트매터 파싱**, **포스트 상세 페이지 연결**이
정상적으로 동작하는지 확인하기 위해 만든 테스트 포스트입니다.

## 확인한 것

- 제목과 날짜가 카드에 노출되는지
- 태그와 카테고리가 상세 페이지에 반영되는지
- 본문에 코드 블록이 문제없이 렌더링되는지

```ts
const message = "MDX is working";
console.log(message);
```

지금처럼 작은 글부터 쌓아두면 디자인을 다듬을 때도 실제 콘텐츠를 기준으로 판단할 수 있습니다.
